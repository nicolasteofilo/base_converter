# Conversão de Bases
Aplicação desenvolvida em Java para conversão de números entre bases numéricas de 2 a 36.

## Objetivo do projeto
Este projeto tem como objetivo aplicar conceitos fundamentais de lógica de programação e sistemas de numeração, implementando manualmente o processo de conversão entre diferentes bases.
Também visa exercitar validação de dados, tratamento de exceções e manipulação de números inteiros de grande porte utilizando `BigInteger`.

## Sobre o projetos
O sistema permite:

- [x] Conversão entre bases de 2 a 36  
- [x] Suporte a dígitos de 0–9 e letras A–Z  
- [x] Manipulação de números de grande magnitude  
- [x] Validação de entradas com tratamento de exceções personalizadas

Este trabalho foi desenvolvido como atividade acadêmica da **FATEC Ribeirão Preto – São Paulo**.

## Requisitos para execução

- Java 17
- IDE compatível com Java (Eclipse, IntelliJ, etc.)

## Execução

1. Clonar o repositório

```bash
git clone https://github.com/nicolasteofilo/base_converter.git
```

2. Importar o projeto em uma IDE Java (Eclipse, IntelliJ, etc.).
3. Executar a classe Main.
